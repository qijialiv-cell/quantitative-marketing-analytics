# HCT Digital Marketing, Analytics & AI+ Lab

Static HTML learning lab for the HCT Digital Marketing, Marketing Analytics and AI+ program.

## Local Preview

```bash
cd apps/hct-lab-static
python3 -m http.server 8080
# Open http://localhost:8080
```
